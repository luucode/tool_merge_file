Libraries need to be installed

pip3 install glob2
pip install pandas
pip install openpyxl


How to use:

Before running delete all files in folder file_csv, file_excel, output_csv, output_excel

+ In case of running need merge file csv
S1: Put all the csv files that need to be merged in folder file_csv
S2: Let's run file func_merge_csv.py

+ In case of running need merge file excel
S1: Put all the excel files that need to be merged in folder folder file_excel
S2: Let's run file func_merge_excel.py
